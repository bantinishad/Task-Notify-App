# Task Notify 

Task Notify is a modern Android Task Reminder application built using **Kotlin** and **XML**.  
The app helps users create, manage, and track daily tasks with reminder notifications, priority levels, recurring schedules, and beautiful UI animations.

---

##  Features

 Add, Edit & Delete Tasks
 Task Reminder Notifications
 Recurring Tasks (Daily / Weekly / Monthly)
 Priority Levels (Low, Medium, High)
 Task Categories
 Mark Tasks as Completed
 Dashboard Statistics
 Beautiful UI with Animations
Local Storage using SharedPreferences + Gson
 Notification Channel Support

---

## 🛠 Built With

- Kotlin
- XML Layouts
- RecyclerView
- AlarmManager
- BroadcastReceiver
- SharedPreferences
- Gson
- Material Design Components

##  Features Overview

### 📝 Task Management

Users can create tasks with:
- Task Name
- Category
- Priority
- Reminder Time
- Recurring Type

###  Smart Notifications

- Uses AlarmManager
- Sends reminder notifications at selected time
- Supports recurring reminders

###  Statistics Dashboard

Shows:
- Total Tasks
- Completed Tasks
- Pending Tasks
- High Priority Pending Tasks
